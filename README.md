# JupiHealth

Додаток, що допоможе дізнатися стан ваги за Індексом Маси Тіла та не забувати про приймання ліків.

Можливості JupiHealth:
- запис параметрів тіла (зріст та вага);
- запис віку та імені користувача;
- обрахунок індексу маси тіла (далі – ІМТ);
- визначення оптимального ІМТ користувача;
- візуальну демонстрацію зміни показника ІМТ;
- встановлення цільового значення ІМТ користувачем;
- визначення, чи є зазначений користувачем ІМТ як цільовий – допустимим та здоровим для нього;
- встановлення графіку прийому ліків;
- відслідковування кількості випитої води за день.
